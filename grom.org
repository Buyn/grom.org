(defn form-group-item
  [{:keys [id type value values]}]
  (let [chenge-value-to-atom
          #(swap! values
                  assoc
                    (keyword id)
                    (.. % -target -value))]
    [:div.form__group
   [:label.form__label {:for id} id]
   [:input.form__input {:type type
                        :id id
                        :value value
                        :on-change chenge-value-to-atom}]]))

(defn form-sold-out
  [values]
  (let [toggle-checkbox

(defn gig-editor-toggle
  ([app] (swap! app update-in [:gig-editor :modal] not))
  ([app status]
         (swap! app assoc-in [:gig-editor :modal] status)))


(defn gig-editor-open
  ([app]
   (gig-editor-toggle app true))
  ([app {:keys [id title desc price img sold-out] :as el}]
    (swap! app
          assoc-in [:gig-editor :values]
          { :id       (or id (str "gig-" (random-uuid)))
            :title    (str/trim title)
            :desc     (str/trim desc)
            :img      (str/trim img)
            :price    (js/parseInt price)
            :sold-out sold-out})
   (gig-editor-toggle app true)))

(defn gig-editor-new
  [app]
  (gig-editor-open app

                    :title "" :desc "" :price 0 :img "" :sold-out false}))

**** свернуть navigation в hamburger menu
Самое важное для мобильной версии — **свернуть navigation в hamburger menu**. Хорошая новость: это можно сделать **почти без изменений структуры**, используя responsive-классы из Tailwind CSS и маленькое состояние в Reagent.

Я покажу **самый безболезненный вариант** — добавляется примерно **20 строк кода**, а остальная разметка остаётся почти такой же.

---
**** # 1. состояние меню

В Reagent добавим атом:

```clojure
(defonce menu-open? (reagent/atom false))
```

---
**** # 2. кнопка hamburger

Добавим кнопку, которая видна **только на мобильных**.

```clojure
[:button
 {:class "md:hidden text-3xl"
  :on-click #(swap! menu-open? not)}
 "☰"]
```

`md:hidden` означает:

```
mobile → показываем
desktop → скрываем
```

---
**** # 3. navigation для desktop

Твоя текущая навигация почти не меняется:

```clojure
[:nav {:class "hidden md:flex space-x-6 font-medium"}
 [:a {:href "#" :class "hover:text-ueava-sand"} "Home"]
 [:a {:href "#"} "About"]
 [:a {:href "#"} "Resources"]
 [:a {:href "#"} "Membership"]
 [:a {:href "#"} "Conferences"]]
```

`hidden md:flex` означает:

```
mobile → скрыта
desktop → обычное меню
```

---
**** # 4. mobile menu

Теперь добавляем выпадающее меню:

```clojure
(when @menu-open?
  [:div
   {:class "absolute top-full left-0 w-full bg-white shadow-md md:hidden"}

   [:div {:class "flex flex-col p-6 space-y-4"}

    [:a {:href "#"} "Home"]
    [:a {:href "#"} "About"]
    [:a {:href "#"} "Resources"]
    [:a {:href "#"} "Membership"]
    [:a {:href "#"} "Conferences"]]])
```

*** как это сделать?
**** плавное выезжание меню сверху
Сделаем **плавное выезжание меню сверху** с помощью transition из Tailwind CSS.
В Reagent это очень просто: мы не скрываем меню через `when`, а **оставляем его в DOM** и меняем классы.

Так браузер сможет **анимировать появление**.

---
**** # 1. идея

Меню будет менять состояние:

```
закрыто:
opacity 0
translateY(-20px)

открыто:
opacity 100
translateY(0)
```

Tailwind умеет анимировать это через `transition`.

---
**** # 2. мобильное меню с анимацией

Заменим блок меню на такой:

```clojure
[:div
 {:class (str
          "absolute top-full left-0 w-full bg-white shadow-md md:hidden
           transform transition-all duration-300 ease-out "

          (if @menu-open?
            "opacity-100 translate-y-0"
            "opacity-0 -translate-y-4 pointer-events-none"))}

 [:div {:class "flex flex-col p-6 space-y-4"}

  [:a {:href "#"} "Home"]
  [:a {:href "#"} "About"]
  [:a {:href "#"} "Resources"]
  [:a {:href "#"} "Membership"]
  [:a {:href "#"} "Conferences"]]]
```

---
**** # 3. кнопка переключения

Кнопка остаётся простой:

```clojure
[:button
 {:class "md:hidden text-3xl"
  :on-click #(swap! menu-open? not)}

 (if @menu-open?
   "✕"
   "☰")]
```

---
**** # 4. что делает Tailwind

Эти классы:

```
transition-all
duration-300
ease-out
```

говорят браузеру:

```
анимируй изменения 300ms
```

А эти:

```
translate-y-0
-translate-y-4
```

двигают меню.

---
**** # 5. результат

Нажимаем ☰

```
tap ☰
↓

```

Закрываем:

```
tap ✕
↓
меню плавно исчезает
```

Анимация занимает **0.3 секунды** — выглядит очень естественно.

```
bg-[url(...)]   → фон
bg-repeat       → повтор
bg-[length:300px] → размер паттерна
```

Tailwind умеет повторять background-image через `bg-repeat` utilities ([Tailwind CSS][2]).
